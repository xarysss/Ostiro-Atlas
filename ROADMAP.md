# Roadmap

## V0.1 - Fondation exécutable

- [x] Monorepo TypeScript et coque desktop Tauri.
- [x] Contrats de fiabilité, provenance et audit.
- [x] Moteur financier isolé et tests initiaux.
- [x] Schéma SQLite et compte de démonstration.
- [x] Dashboard, portefeuille et santé des données.
- [x] Pipeline CI pour les installateurs Windows.

## V0.2 - Saisie et import

- [x] Accueil, profils locaux et onboarding guidé.
- [x] Personnalisation des modules et dashboard de démarrage.
- [x] Mode développeur et scénarios de démonstration.
- [ ] CRUD complet des comptes, actifs et transactions.
- [ ] Assistant CSV : aperçu, encodage, mapping et validation.
- [ ] Détection de doublons et modèles d'import persistants.
- [ ] Reconstitution du PRU avec résolution des anomalies.
- [ ] Export CSV/JSON et sauvegarde locale versionnée.

## V0.3 - V1 desktop locale

- [ ] Dividendes, frais et performance détaillée.
- [ ] Chiffrement SQLCipher opt-in et sauvegarde `.owb` chiffrée.
- [ ] Verrouillage par mot de passe maître et inactivité.
- [ ] Signatures Windows et mises à jour GitHub opt-in.
- [ ] Documentation utilisateur et tests E2E des parcours critiques.

## Après la V1 stable

- Connexions DSP2 en lecture seule via prestataires réglementés.
- Synchronisation cloud chiffrée et strictement optionnelle.
- Connecteurs de prix publics avec clés fournies par l'utilisateur.
- Linux et macOS, puis mobile si l'usage le justifie.
- Mode famille, société et holding.

## Hors périmètre sans cadre réglementaire

- recommandation personnalisée d'investissement ;
- achat, vente, échange ou conservation de crypto-actifs ;
- initiation de paiement ou passage d'ordres ;
- déclaration fiscale présentée comme officielle.
