# Code de conduite

Nous voulons une communauté accueillante, rigoureuse et respectueuse, quels que soient l'expérience, l'identité, l'origine ou le contexte financier de chacun.

## Comportement attendu

- critiquer les idées et le code, jamais les personnes ;
- expliquer les décisions techniques avec patience ;
- protéger strictement la vie privée et ne jamais demander de données réelles ;
- déclarer les conflits d'intérêts, notamment avec un fournisseur financier ;
- accepter les corrections factuelles et réglementaires.

Le harcèlement, la discrimination, la divulgation de données personnelles et la promotion financière trompeuse ne sont pas tolérés. Les incidents peuvent être signalés par le canal privé indiqué dans `SECURITY.md`. Les mainteneurs peuvent avertir, masquer du contenu ou exclure un participant selon la gravité.

Ce texte s'inspire du Contributor Covenant 2.1.
